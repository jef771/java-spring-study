# java-spring-study
A small and simple project used to study spring

It contains brief explainations about:
- Get, Post, Delete, and Put mappings
- Autowired
- Service
- RestController
- Transactional
- Repository
- Query
- Configuration
- Bean

# Usage
- Install Intellij if you don't have
- Open up the project
- Build some tests!


´´ Tip: use postman to test put and delete
